There is no "my" in open source

If you use Node, you've probably been following this week's story between Azer Koçulu, Kik and npm. A brief rundown: Azer made an npm module called "kik", which shares its name with a company. Kik asked him to rename the module, and Azer refused, so npm intervened and reassigned it to Kik.
